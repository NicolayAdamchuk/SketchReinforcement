Запуск приложения и его результаты

Для выполнения приложения следует щелкнуть на кнопке Эскизы арматуры.

Примечание: до выполнения команды файл проекта должен быть сохранен.

 В открывшемся диалоговом окне, можно активировать некоторые режимы:

  • Создать эскизы для всех стержней  - создаются эскизы для всех стержней проекта
  • Создать эскизы для выбранных стержней - создаются эскизы для стержней, выбранных на текущем виде. После выбора стержней следует нажать кнопку Готово
  • Создать эскизы для стержней текущего вида - создаются эскизы для стержней текущего вида
  • Создать эскизы для выбранного раздела - создаются эскизы для стержней определенного раздела (если разделы были назначены)


Кроме того, можно:  

  • включить флаг Показать длину прямого участка крюков - для отображения длины прямого участка крюка
  •  выбрать тип и размер шрифта


Вкладка Свободные формы.

Доступна опция Показать угол для свободной формы. На эскизе стержня будут показаны углы загиба.


Вкладка Стандартные формы.

       Здесь группа параметров позволяет создавать эскизы арматуры на основе составных стержней. Эти параметры используются только для второго способа создания эскизов - на основе стандартных форм. Составные стержни формируются условно (в модели проекта изменения не производятся) с учетом назначенной максимальной длины стержня и нахлеста стержней. В тех случаях, когда длина стержня превышает максимальную производится разделение стержня на отдельные участки так, чтобы длина участка не превышала максимум.

Такие эскизы создаются для ограниченного числа стандартных форм. Опция работает для форм 1,11,12,13,21 (нормы BS8666:2005) и 1,2,3,4 (нормы ГОСТ 21-501). Например,

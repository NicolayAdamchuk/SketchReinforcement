Running add-in and its results

To run the application you must click on the button Sketches reinforcement. Note: project file should be saved before to run.

In the dialog box that opens, you can activate some modes:

  • Create sketches for all rebars - create sketches for all rebars of the project
  • Create sketches for some rebars - sketches are created for rebars selected in the current view. After selecting the rebars, click Finish
  • Create sketches for rebars of current view - create sketches for rebars of the current view
  • Create sketches for some partition - sketches are created for the rebars of a specific partition (if partitions have been assigned)


You can also:

  • switch on flag Show hooks length (straight segement) - to display the length of the straight section of the hook
  • select the font type and size.


Tab Free shape

Option available Show angle for free form. A bar sketch will show its bend angles.


Tab Standard shape

The group`s options  allows to create the sketches reinforcement from composite bars. These options are used only for the second method of sketching — based on standard shapes. Composite bars are conditionally generated (no changes in the project model) based on the assigned maximum bar length and bar overlap. In cases where the length of the bar exceeds the maximum, the bar is divided into separate sections so that the length of the section does not exceed the maximum.

These sketches are created for a limited number of standard shapes. The option works for forms 1,11,12,13,21 (code BS8666: 2005) and 1,2,3,4 (gode GOST 21-501). For instance,

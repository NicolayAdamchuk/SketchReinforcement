Getting sketches using standard pictures

The method uses a ready pictures library with standard rebar shapes. System family of rebar shape should contain a reference to a standard picture for performing the method. A link is created as follows:

- select the rebar in the project or the rebar shape of the system family in the Project Browser

  • open Family Editor
  • open Family Types, click on button
  • for parameter Shape Image click on button

  • in the dialog box Manage Images click on button Add
  • open the folder with the pictures library, which is located in "... Users \ Common \ Shared Documents \ Autodesk \ Downloaded content \ Ar-Cadia \ SketchReinforcement"

 

  • open the folder with the appropriate standard. In this version are two standard BS8666-2005 and Gost21-501. Information on the standard figures is lower.
  • select the appropriate standard picture, click button Add
  • now parameter Shape Image contains a link to the desired picture

 

- нажмите кнопку ОК и выполните загрузку семейства в проект, нажав на кнопку (измененный файл семейства можно не сохранять)

- press on OK button and follow the download of the family in the project by pressing button  (you can not save the modified of the family file)

- this procedure is performed once (for a project file or a template file) for the current rebar shape. Now every time when using this form of the rebar will be used by the downloaded picture. An example is shown below:
